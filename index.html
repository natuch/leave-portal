ตั้งค่าการเชื่อมต่อ API

เชื่อมโยงระบบบันทึกวันลากับ Google Sheets และผู้ช่วยสมองกล Gemini AI เพื่อเพิ่มพลังการประมวลผลภาษาธรรมชาติ

การเชื่อมต่อ Google Sheets API

ระบุ URL ที่ได้จากการกด "Deploy > Web App" ใน Apps Script ของ Google Sheets (ดูขั้นตอนการทำในฝั่งคู่มือช่วยเหลือได้เลยครับ)

การเปิดใช้ฟีเจอร์เสียงและภาษาอัจฉริยะ (Gemini AI Key)

ระบุ API Key เพื่อเปิดใช้ระบบแปลความหมายภาษาไทยและอังกฤษอัจฉริยะ และ แชทบอทตอบคำถามบน Dashboard สามารถขอ API Key ได้ฟรีผ่านทาง Google AI Studio

คู่มือเชื่อมโยง Google Sheet ใน 3 ขั้นตอน

  • 1
    สร้างไฟล์ Google Sheets ขึ้นมา แล้วเลือกแท็บด้านบน ส่วนขยาย (Extensions) > Apps Script
  • 2
    คัดลอกโค้ดสคริปต์ด้านล่างนี้ ไปลบโค้ดเดิมทั้งหมดในหน้าต่าง Apps Script แล้ววางโค้ดสคริปต์นี้ลงไปแทน
    google_apps_script.js
    /* กำลังโหลดโค้ดสคริปต์... */
  • 3
    กดปุ่มบันทึก (แผ่นดิสก์) จากนั้นเลือกปุ่ม การทำให้ใช้งานได้ (Deploy) > การจัดการทำให้ใช้งานได้ (New Deployment)

    * เลือกประเภทเป็น "เว็บแอป (Web App)"
    * กำหนดสิทธิ์ผู้เข้าใช้งานแอปเป็น "ทุกคน (Anyone)"
    * กดปุ่ม Deploy ให้สิทธิ์การเชื่อมต่อ แล้วคัดลอก "URL ของเว็บแอป (Web App URL)" มาวางในช่องตั้งค่าด้านข้างได้เลยครับ!
